Angular Playground - Projeto de introdução

O objetivo deste projeto é propor desafios para introduzir o dev à stack de tecnologias utilizadas no desenvolvimento front-end como Angular, TypeScript, RxJS, NgRx, SASS e Bootstrap e exercitar o desenvolvimento de soluções.

Descrição do desafio:

O desafio consiste em criar uma página que consulte a API do Github para listar todos os repositórios públicos da globo.com e, ao clicar em um repositório, listar algumas informações do repositório como nº de stars, forks e lista de commits.
